Radium Lead Form

This is a simple application you can fork and use to generate your own lead capture form to work with Radium.

The form is setup with some basic information. Your job to create a form that make sense with the Radium api. You can add more fields to mix and match emails/phones numbers/addresses/extra fields and notes to get it how you want it. When have generated the form the way you want it, press "Test". This shows you the generated JSON that will be sent to Radium. You can test as many times as you want.

The submitting the form will send the data to Radium so you can test out how it works. You can use this as a base to create something for your own website.
